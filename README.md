# Simple Node.js Web Application

This is a simple web application built with Node.js, Express, HTML, CSS, and JavaScript. It serves static files and demonstrates basic web development concepts.

## Project Structure

```
simple-node-webapp
├── public
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Styles for the web application
│   └── js
│       └── main.js       # Client-side JavaScript
├── src
│   └── server.js         # Entry point for the Node.js server
├── package.json          # npm configuration file
├── .gitignore            # Files and directories to ignore by Git
└── README.md             # Project documentation
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd simple-node-webapp
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the server**:
   ```
   node src/server.js
   ```

4. **Access the application**:
   Open your web browser and go to `http://localhost:3000`.

## Usage

This application serves a simple web page with client-side interactivity. You can modify the HTML, CSS, and JavaScript files to customize the application as needed.
